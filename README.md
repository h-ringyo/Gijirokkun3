# Gijirokkun3

## 概要
議事録などの音声ファイルからの文字起こしを意識した再生アプリです。

・mp3とwaveファイルの再生

・再生速度の可変

・再生位置のスキップ送りと戻し

・再生位置にtagとコメントをつける

## 特記事項
